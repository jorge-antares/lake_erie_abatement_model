Lake Erie Abatement Model

Model Overview

The Lake Erie Abatement Optimization model addresses phosphorus pollution through two complementary approaches:

1. Target-Based Optimization

Objective Function:
$$\min f(\mathbf{x}, \mathbf{v}) = \mathbf{x}^T \mathbf{A} \mathbf{x} + \mathbf{b}^T \mathbf{v}$$
Subject to:
$$\mathbf{S} \mathbf{x} + \mathbf{W} \mathbf{v} \geq \mathbf{z}_{target}$$ $$\mathbf{x} \geq 0$$ $$\mathbf{v} \in \{0,1\}^{I}$$
Where:
$$\mathbf{W} = \mathbf{S} \times \mathbf{L} \times \mathbf{F}$$

2. Budget-Constrained Optimization

Objective Function:
$$\max g(\mathbf{x}, \mathbf{v}) = \mathbf{\alpha}^T (\mathbf{S} \mathbf{x} + \mathbf{W} \mathbf{v})$$
Subject to:
$$\mathbf{x}^T \mathbf{A} \mathbf{x} + \mathbf{b}^T \mathbf{v} \leq B$$ $$\mathbf{x} \geq 0$$ $$\mathbf{v} \in \{0,1\}^{I}$$

Variable Definitions

Variable Description Units Dimension
$\mathbf{x}$ Agricultural abatement by region tonne/year $\mathbb{R}^{r}_+$
$\mathbf{v}$ Binary variables for WWTP investments unitless $\{0,1\}^{I}$
$\mathbf{A}$ Agricultural cost matrix (quadratic) Million CAD/(tonne2 year) $\mathbb{R}^{r \times r}$
$\mathbf{b}$ WWTP maintenance costs Million CAD/(thousand m3 year) $\mathbb{R}^{I}$
$\mathbf{S}$ Steady state mass balance matrix ppb/tonne $\mathbb{R}^{r \times r}$
$\mathbf{L}$ Matrix that links WWTPs to regions unitless $\mathbb{R}^{r \times I}$
$\mathbf{F}$ Diagonal matrix of WWTP effective abatement tonne/year $\mathbb{R}^{I \times I}$
$\mathbf{\alpha}$ Relative weight vector for regions unitless $\mathbb{R}^{r}$

Lake Erie Regions

The model considers six distinct regions of the Lake Erie system:

  1. SCR - St. Clair River
  2. LSC - Lake St. Clair
  3. DR - Detroit River
  4. WB - Western Basin
  5. CB - Central Basin
  6. EB - Eastern Basin
