Lake Erie Abatement Model



The Lake Erie Abatement Model (LEAM) is a phosphorus abatement optimization model for the Lake Erie basins. This tool helps optimize agricultural and wastewater treatment investments to reduce phosphorus concentrations in Lake Erie.

Model Overview

The optimization model minimizes the cost of phosphorus abatement while meeting target concentration reductions across six Lake Erie regions.

  • Decision Variables:
  • • Agricultural abatement
  • • WWTP investments
Lake Erie Regions

The model covers six regions of Lake Erie:

  • • St. Clair River (SCR)
  • • Lake St. Clair (LSC)
  • • Detroit River (DR)
  • • Lake Erie Western Basin (WB)
  • • Lake Erie Central Basin (CB)
  • • Lake Erie Eastern Basin (EB)
